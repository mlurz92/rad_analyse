Über RAD Analyse Tool

Das RAD Analyse Tool ist ein professionelles Werkzeug zur Analyse radiologischer Untersuchungsdaten. Es ermöglicht das Hochladen von JSON-Dateien, das Filtern der Daten nach verschiedenen Kriterien und die Anzeige umfassender Statistiken. Das System läuft lokal auf einem Raspberry Pi 4 und bietet eine moderne, responsive Benutzeroberfläche mit dunklen Farbtönen und sanften Glüheffekten.

Entwickelt von mlurz, bietet das Tool eine einfache Handhabung und eine effiziente Datenverarbeitung, um Ihre radiologischen Daten optimal auszuwerten.
